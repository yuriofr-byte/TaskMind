# TaskMind - Assistente Diario de Tarefas com IA

Assistente inteligente de produtividade que organiza e prioriza as tuas tarefas do dia usando Inteligencia Artificial.

## Funcionalidades

- Adicionar tarefas com titulo, descricao e prazo
- IA analisa e prioriza as tarefas do dia
- Lista todas as tarefas com estado
- Marcar tarefas como concluidas
- Envio de resumo diario por email

## Tecnologias

- Python 3
- Google Gemini AI
- Requests
- SMTP (envio de email)

## Como usar

1. Instala as dependencias: pip3 install requests
2. Configura as variaveis de ambiente:
   export GEMINI_API_KEY="a_tua_chave"
   export EMAIL_REMETENTE="teu_email@gmail.com"
   export EMAIL_SENHA="tua_app_password"
   export EMAIL_DESTINO="teu_email@gmail.com"
3. Executa: python3 taskmind.py
